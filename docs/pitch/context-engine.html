Where this goes

The gap nobody's filled: retrieving a decision, not a topic.

Every memory tool — including this one, today — retrieves by subject. Ask “what did I decide was good enough to pitch,” and topical search hands back things about pitching, never the moment you said yes, that one.

That's a speech-act query, and embeddings are structurally bad at it. The fix is extraction at ingest — typing each conversation into decisions, endorsements, commitments, open questions — so a stance becomes a filter, not a fuzzy guess.
