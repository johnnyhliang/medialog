- **EasyClangComplete:** Provides robust C/C++ code completion.
- **LSP-clangd:** Integrates clangd, a language server for C/C++, offering features like code completion, diagnostics, and more via the Language Server Protocol (LSP).
- **C++ Snippets:** Offers a collection of C++ code snippets for faster coding.
- **C Improved:** Enhances the syntax definition for C/C++/Objective-C languages in Sublime Text, primarily focused on pure C but affecting other C-family languages.
- **C++ Starting Kit:** Combines and improves C, C++, and C++11 syntax definitions, and provides a custom build system for the g++ compiler.
- **CppFastOlympicCoding:** Designed specifically for competitive programming in C++, offering features like a test manager, stress testing, CppLint integration, a debugger (for OSX), and class completions.
- **SublimeLinter:** A general-purpose linting framework that can be configured with C++ linters to check for code style and potential errors.
