Material for Vim / Neovim learning around the internet:
→  [`:h user-manual`](<https://neovim.io/doc/user/usr_toc.html#user-manual>) and the `:Tutor` (Neovim) / `vimtutor` commands :pepesmile:
→ VSCode Vim + Learn Vim extensions for VS Code / Any other extension for your editor of choice
→ [Prime's playlist on Vim motions](<https://www.youtube.com/playlist?list=PLm323Lc7iSW_wuxqmKx_xxNtJC_hJbQ7R>)
→ [kickstart.nvim](<https://github.com/nvim-lua/kickstart.nvim>) as a great base for a configuration (other templates made by members of the discord like [`barebone.nvim`](<https://github.com/leet0rz/barebone-nvim>)( @nilsso ) and [`kickballs.nvim`](<https://github.com/AlphaKeks/kickballs.nvim>) ( @.alphakeks ) are good options too)
→ [How2nvim](<https://github.com/AlphaKeks/how2nvim>) (a guide on using Neovim made by @.alphakeks )
→ [LSP-Zero](<https://github.com/VonHeikemen/lsp-zero.nvim#how-to-get-started>) for both new and regular users (you might [not need it](<https://github.com/VonHeikemen/lsp-zero.nvim/blob/v3.x/doc/md/guides/you-might-not-need-lsp-zero.md>))
→ [Understanding Neovim Youtube playlist](<https://www.youtube.com/playlist?list=PLx2ksyallYzW4WNYHD9xOFrPRYGlntAft>) (ongoing)
→ [Neve - An advanced and custom config for NixVim users](<https://github.com/redyf/Neve>) - A bit advanced, but oriented to Neovim + NixOS users who want to get involved

Cool websites to visit:
- <https://vim.how/> - search for vim commands online
- <https://dotfyle.com/> - To look at many other configs and plugins
